# AmoyBusEnglish

这是厦门公交英译小组的英译项目。厦门公交英译小组计划将全市的公交站名均翻译为英语。如果您希望在展望视频中使用本数据库的材料，可以参见[这份指导](https://github.com/EveningSmoke/AmoyBusEnglish/blob/main/pov-attribute-suggestion.md)。

厦门公交英译小组由鹭岛交通迷社区运作。
